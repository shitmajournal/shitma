# Shitma - The academic node for males

SHITMA（中文名：矢么；俗名：💩特漫）是一个专用于回收论文、制造粪肥的学术节点。它继承SHITMA的核心堆肥哲学，同时针对不同学术平台产出的独特气质进行了定向优化，致力于将科研过程中产生的硬核废料转化为滋养学术沃土的专属肥料。


SHITMA维持男性投稿的原则，我们相信这是时代的选择。


SHITMA确保每一篇匿名投稿的论文都能得到公正的分解，无论作者来自哪所高校、哪个学科。堆肥过程全程录像，可供投稿人查阅，但最终肥料将混合使用，体现学术共产精神。

## 如何投稿

由于面向未来堆肥的特性，SHITMA采用Typst作为论文模板。我们希望您能按照以下模板进行投稿，以便我们更好地处理您的论文。以LaTeX、Markdown等格式撰写的论文暂不接受。


### 投稿模板：[shitma-template](https://github.com/shitmajournal/shitma-template)
